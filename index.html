Publications

RAHMAN, T., KHANOM, A. Z., FARZANA, S. M., KHAN, S.M.S., & ALAM, A. (2019, JULY). ANALYSIS OF LINEAR AND NON-LINEAR CLASSIFIERS IN IMBALANCED DATA TO PREDICT DIABETES INDUCED COMPLICATIONS. IN PROCEEDINGS OF THE 15TH INTERNATIONAL CONFERENCE ON MACHINE LEARNING AND DATA MINING, MLDM 2019, VOL. I.
Thesis: Prediction of Diabetes Induced Complications Using Different Machine Learning Algorithms (PDF)

A system that predicts the risk of diabetes related complications like nephropathy and cardiovascular diseases using classification algorithms like Logistic Regression, SVM, Decision Tree, Random Forest, etc. The data is first preprocessed and then PCA is applied to reduce the number of dimensions. SMOTE is also applied for imbalanced data. The system was implemented in Python using the Scikit-Learn library. The thesis work was done in a group of 3 individuals under the supervision of a senior faculty member.
